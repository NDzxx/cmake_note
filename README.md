安装win什么的不说了，直接下exe即可
ubuntu:  

```shell
 #cmake安装
 sudo apt-get install cmake  
 
 #cmake gui安装
 sudo apt-get install cmake-qt-gui  
```

```
 #如果需要卸载cmake
 apt-get autoremove cmake
```
